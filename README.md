## Getting Started

### Installation For Local Server

1. Clone the repo
   ```sh
   git clone https://github.com/iainchun/stormhacks2024.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Deploy Server
   ```sh
   node app.js
   ```
